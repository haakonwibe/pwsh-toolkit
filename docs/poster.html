✓ PWSH-TOOLKIT 
# Profiles/config.psd1 — one declarative file
@{
    Prompt        = 'OhMyPosh'   # or 'Custom' / 'Default'
    OhMyPoshTheme = 'Random'     # fresh prompt every shell

    ExtraJumpFolders = @(        # j — bookmarked folders
        @{ Label = 'Work'; Path = 'D:\Work' }
    )
    ProjectRoots = @('C:\GitHub')  # prj — jump to any repo

    DisableStartupTips = $false      # the rotating tip
    Features = @{ DisableM365 = $false }  # feature toggles
}
# same setup on every machine — j · peek · df · winup

What's in the box?

Config-driven loader

One config.psd1 picks the prompt (Oh My Posh, custom, default), OneDrive org, jump folders, and feature toggles. ToolkitRoot and OneDriveOrg auto-detect from environment.

Folder jumper (j)

Interactive picker with digit shortcuts 1-9, arrow nav, or direct fuzzy match: j prog. Renders on the alternate screen buffer — your scrollback survives. Browser-style jb/jf history.

Archive previewer (peek)

Extracts any archive to a temp folder and drops you inside. WinRAR for .rar, 7-Zip for the rest, Expand-Archive fallback. Plus -List / -Active / -Clean.

Smart install, clean uninstall

Symlink if you can (admin / Developer Mode), dot-source stub otherwise — both live-edit, no fuss. Optional Oh My Posh + Nerd Font + Terminal-Icons in one prompt. install.ps1 -Uninstall backs out cleanly and restores your old profile.

AI Downloads tagger

tagdl writes Claude-generated, BBS-style descriptions to NTFS streams plus a portable index CSV. Browse with dird or newest-first fr. ~$0.001 per file.

Per-machine + per-host overrides

Machines/<NAME>.ps1 for per-machine tweaks, Hosts/<HostName>.ps1 for per-host (VS Code, Terminal, ISE) — dot-sourced after Common, with full access to everything the loader defines.

Oh My Posh, randomized

Update-PoshThemes pulls the full ~120-theme gallery. Set OhMyPoshTheme = 'Random' for a fresh prompt each shell — it names the one it rolled so you can pin it. Browse and switch live with Set-PoshTheme.

Built-in discovery

toolkit lists every command grouped by area with a one-line synopsis — the Get-Command for a profile that isn't a module. Every command also has Get-Help, plus a rotating startup tip.

Shell & quick commands

  • ll — detailed directory table
  • la — list incl. hidden & system
  • lh — only hidden & system entries
  • touch — create / bump timestamp
  • which — what backs a command
  • winup — winget upgrade picker

Navigation

  • j — bookmarked-folder jumper
  • jb / jf — jump back / forward
  • prj — jump to any git repo
  • mkcd — make dir & cd into it
  • up — go up N parent dirs
  • .. / ... — up one / two dirs
  • docs · desktop · downloads — OneDrive shortcuts
  • onedrive · home — root / user profile

Discovery

  • tip — a random profile tip
