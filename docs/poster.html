What's in the box?

Config-driven loader

One config.psd1 picks the prompt (Oh My Posh, custom, default), OneDrive org, jump folders, and feature toggles. ToolkitRoot and OneDriveOrg auto-detect from environment.

Folder jumper (j)

Interactive picker with digit shortcuts 1-9, arrow nav, or direct fuzzy match: j prog. Renders on the alternate screen buffer — your scrollback survives. Browser-style jb/jf history.

Archive previewer (peek)

Extracts any archive to $env:TEMP\peek\… and jumps you in. Dispatches to WinRAR for .rar, 7-Zip for everything else, Expand-Archive for .zip as a fallback. peek -List, -Active, -Clean for the obvious variants.

Smart install (no admin)

Probes for symlink capability — admin or Windows Developer Mode → symlink, otherwise a dot-source stub. Both give you live-edit; neither requires you to think about it. Plus optional Oh My Posh + Meslo Nerd Font + Terminal-Icons in one prompt.

AI Downloads tagger

tagdl scans ~\Downloads, calls Claude Haiku with a structured-output schema, and writes BBS-style descriptions to NTFS Alternate Data Streams plus a portable index CSV. Browse with dird or paged-newest fr. Costs ~$0.001 per file.

Per-machine + per-host overrides

Drop a Machines/<COMPUTERNAME>.ps1 for machine-specific tweaks; Hosts/<HostName>.ps1 for per-host (VS Code, Windows Terminal, ISE). Both dot-source after the Common helpers — full access to every function, alias, and $script: variable the loader defines.
